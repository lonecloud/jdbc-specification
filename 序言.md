本文是对之前规范的内容进行一些补充并取代一些内容，主要涉及如下几个方面：

- “JDBC: A Java SQL API”
- “JDBC 2.1 API”
- “JDBC 2.0 Standard Extension API”
- “JDBC 3.0 规范”

本文档介绍了JDBC API的一系列新功能，并结合了各种规范改进，这些改进侧重于JDBC 3.0 API或之前引入的功能。 我们会对在基于JDBC 3.0 API更新或修订的部分，将会在JDBC 4.2 API中进行标记。
